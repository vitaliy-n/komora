# Комора

**Комора** — це веб-додаток для домашнього консервування, обліку запасів та управління продуктами. Створений для тих, хто любить заготовляти смачні речі на зиму: варення, компоти, маринади, соуси, салати, соки, джеми, аджику та інші домашні заготовки.

Додаток працює прямо в браузері як **Progressive Web App (PWA)** та зберігає дані локально за допомогою IndexedDB (Dexie.js).

## Можливості

- **Довідник продуктів** — понад 850 вбудованих продуктів з інформацією про термін зберігання, температуру, вологість, сумісність, ознаки псування та сезонність.
- **Рецепти** — вбудована колекція рецептів для консервування, сортування за категоріями, пошук за інгредієнтами.
- **Закрутки** — облік домашніх заготовок: що зроблено, коли, з яких продуктів, термін придатності.
- **Запаси / інвентар** — облік продуктів вдома з відстеженням термінів придатності.
- **Календар сезонності** — візуальний календар, який показує, які продукти зараз в сезоні.
- **PWA** — можна встановити на телефон або комп'ютер, працює офлайн.
- **Синхронізація вбудованих даних** — при оновленні додатка нові продукти та рецепти автоматично додаються до локальної бази.

## Технології

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Zustand
- Dexie.js (IndexedDB)
- vite-plugin-pwa
- lucide-react

## Локальний запуск

```bash
npm install
npm run dev
```

## Збірка

```bash
npm run build
```

## Дані

Вбудовані дані зберігаються у вихідному коді:

- `src/data/products/` — продукти за категоріями
- `src/data/recipes-db.ts` — рецепти
- `src/data/categories.ts` — категорії рецептів

Користувацькі дані (закрутки, запаси, налаштування) зберігаються локально в IndexedDB браузера.
